Car search application
========================

How to run it?
--------------

Build with automated acceptance tests (Firefox needed):

`./gradlew clean build`

Embedded Jetty:

`./gradlew run`


Technologies used:
------------------

- Java 7
- ElasticSearch 
- AngularJS
- Spring Core, MVC

### tested with:
- JBehave
- Selenium
- TestNG
